Características

Los cursos deben estar construidos sobre un software adecuado, es decir, sistemas web creados específicamente para satisfacer las necesidades necesarias para impartir cursos online.

Optimización

La plataforma permite difundir el aprendizaje de una forma económica en cualquier lugar y en cualquier momento.


Ventajas:

  • Ahorrar costo en transporte.
  • Reducción en el tiempo de aprendizaje.
  • Reduce los costos de producción.
  • Administrar el tiempo de los alumno de manera eficiente.

Diseño Responsive

Permite un diseño responsive que se adapte a todo tipo de tamaños de pantalla y dispositivos, como portátiles y teléfonos móviles.

Procesos Automáticos

Una plataforma virtual necesita estar viva y ser dinámica, sin la necesidad de estar constantemente monitoreando cada uno de sus procesos.

Acerca de

Plataforma Virtual para la Impartición de Cursos y Capacitaciones.

  • 2018

    Proyecto

    La plataforma se realizó como proyecto de residencia profesional en el Instituto Tecnológico de Villahermosa para ser implementado en el Despacho Contable-Fiscal Ramírez, desarrollado por el I.S.C. Adrian Garciliano Jimenez.

  • Habilidades

    Conocimientos aplicados en el desarrollo

    Programación Orientada a Objetos, lenguaje de programación PHP, metodología SCRUM, desarrollo de la Base de Datos MySQL, modelo MVC, Servicio CRON, desarrollo de la interfaz gráfica con CSS y HTML, framework Bootstrap, editor VIM, sistema operativo GNU/Linux Ubuntu y sistema de control de versiones Git.

Página Web de la empresa

Desarrollo de una página web dinámica para la imagen del Despacho Contable-Fiscal Ramírez.



El proyecto de residencia era originalmente solo el sistema para impartir cursos online, pero dada la situación del despacho que no contaba con una pagina web que representara la imagen de este, el L.C.P. Juan José Ramírez Olán me solicito realizar una pagina web en la cual podría subir la información de los servicios que brinda, así como compartir algunas noticias de interés en el ámbito contable fiscal, y de esta forma también implementar acciones que permitan tener un contacto con clientes por medio de la red de Internet.

Sistema Administrativo para la Página Web

Sistema para administrar el contenido de la página web.







Sistema para administrar de manera dinámica el contenido de la pagina web, desde las imágenes hasta el contenido de la información.

El administrador puede modificar los siguientes recursos:

  • Imágenes y texto del Carousel
  • Imágenes e información de los servicios
  • Imagen, información y links de las noticias
  • Leer y responder mensajes recibidos desde la página web
  • Enviar publicidad a todos los suscriptores de la página sobre los diversos servicios, cursos y noticias
  • La información sobre la empresa y contacto
  • La información sobre la empresa y contacto
  • Visualizar Gráficas que muestran información sobre el numero de visitas ya sea por las secciones de la página, por los servicios o cursos ofertados en la página web.
  • CRUD de los elementos antes mencionados

Sistema Administrativo para la Plataforma Virtual

Sistema para administrar el contenido de los cursos y capacitaciones.







Sistema para administrar de manera dinámica el contenido de los cursos y capacitaciones, asi como el manejo y creación de grupos y perfiles de alumno.

El administrador puede modificar los siguientes recursos:

  • Años y periodos
  • Temas (cursos, Capacitaciones o Talleres)
  • Estados de los registros (activos o inactivos)
  • Recursos PDF y Video
  • Información de los Temas, Módulos y Submódulos
  • Grupos de alumnos para un determinado tema
  • Permitir o denegar el acceso a los estudiantes a un tema especifico
  • Listado de los registros
  • Solicitudes de los alumnos para cursar un Tema
  • Envió de correos electrónicos para recuperar contraseñas o notificar cambios en la cuenta
  • Visualización clara y precisa de los elementos en el sistema
  • CRUD de los elementos antes mencionados
  • Procesos automáticos de activación y desactivación de los temas de acuerdo a las fechas establecidas (CRON)
  • Gráficas que muestran el estado actual de la plataforma virtual

Sistema del alumno para la Plataforma Virtual

Sistema para el alumno.







Sistema del alumno en la cual podrá acceder a la información y documentación de los cursos a los que ha sido asignado por el administrador.

El alumno puede realizar las siguientes acciones:

  • Acceder a la información de los cursos
  • Realizar solicitudes para otros talleres o capacitaciones
  • Enviar y recibir correos del administrador
  • Visualizar el estado de los cursos a los que esta asignado
  • Visualizar su información de contacto
  • Inicio de sesión y recuperación de contraseña
  • Registrar una cuenta

Gráficas

Gráficas sobre el estado de la plataforma virtual.







Gráficas para analizar a cerca de los puntos fuertes y débiles de las áreas de la página web, así como de los servicios y cursos; esto permitirá tomar decisiones asertivas que permitan mejorar el servicio.

Procesos Automáticos

Procesos CRON.







Procesos automáticos utilizando el servicio de CRON en el cual se programo un script php para poder realizar procesos programados, estos se encargan de verificar las fechas de inicio y terminación de cursos y módulos y activarlos o desactivarlos inmediatamente; esto para que el administrador pueda enfocarse en tareas de mayor interés y estar revisando las fechas diariamente.

Manuales.

3.- Manual de usuario sistema de cursos administrador : Información sobre la correcta administración del contenido de los cursos y los usuarios.

4.- Manual de usuario sistema de cursos alumno: Información sobre la interfaz del sistema del alumno para poder tomar los cursos a los que se le ha brindado el acceso.
